Copy and paste this entire content into your server/.env file on EC2:

# Server Configuration
PORT=5000
NODE_ENV=production

# Database - SQLite (for production - consider PostgreSQL for scale)

# JWT Configuration - CHANGE THIS TO A SECURE RANDOM STRING!

# File Upload
MAX_FILE_SIZE=5242880
UPLOAD_PATH=./uploads

# Rate Limiting
RATE_LIMIT_WINDOW_MS=900000
RATE_LIMIT_MAX_REQUESTS=100

IMPORTANT: 
1. Change JWT_SECRET to a secure random string!
   You can generate one with: openssl rand -base64 32
   
2. After pasting, save with Ctrl+X, then Y, then Enter
